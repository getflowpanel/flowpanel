flowpanel 1.0.0 — Initial public release.

A typed config becomes a full `/admin` route for your Next.js 15 app. Drizzle or Prisma adapter. Type-safe column inference from your schema (`columns: ["typo"]` errors at the call site with a "Did you mean" suggestion). Server actions with toast + table refresh. Drawer with URL-sync and configurable fields. Dashboards with metric / table / chart widgets, server-prerendered cells, and `ChartBucket`-aware date formatting. ⌘K command palette free with any registered resource. Dark mode that persists across navigations. BullMQ queue UIs via the adapter. Realtime via SSE — memory driver for dev, Redis for prod. Mobile-responsive shell. Theme.components slot registry typed against `FlowpanelComponentSlots`. `flowpanel eject` writes the source into your repo with a version stamp when you outgrow the config layer.
